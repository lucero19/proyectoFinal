¿Qué es Dreamweaver? y ¿Qué es CSS?

Dreamweaver:

Es una aplicación en programa de estudio (basada en la forma de estudio de Adobe Flash) que está destinada a la construcción, diseño y edición de sitios, vídeos y aplicaciones Web basados en estándares. Creado inicialmente por Macromedia (actualmente producido por Adobe Systems) es uno de los programas más utilizados en el sector del diseño y la programación web por sus funcionalidades, su integración con otras herramientas como Adobe Flash y, recientemente, por su soporte de los estándares del World Wide Web Consortium.

Con Dreamweaver se pueden crear archivos como:

  • HTML
  • CSS
  • Javascript
  • XML

¿Qué es CSS?

  • Cascading Style Sheets (CSS) ‘Hoja de estilo en cascada’ es un lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML.
  • CSS separa la estructura de un documento de su presentación.
  • La información de estilo puede ser definida en un documento separado o en el mismo documento HTML.

Evolución de CSS:

El organismo W3C (World Wide Web Consortium), encargado de crear todos los estándares relacionados con la web, propuso la creación de un lenguaje de hojas de estilos específico para el lenguaje HTML

Agregando hojas de estilo:

CSS3 Múltiples estilos un documento

Sintaxis de CSS

CSS tiene una sintaxis sencilla, que usa palabras clave (inglés) para especificar los nombres de las propiedades y la mayoría de los valores. Cada línea terminan con ;.

Estilo-CSS:

Selectores:

Selectores CSS

Definen que elementos HTML serán afectados por las declaraciones

Existen varios tipos de selectores:

  • Elementos
  • Clases
  • ID’s
  • Descendientes
  • Adyacentes (hermanos o siblings)

Selector Universal

El selector universal, selecciona todos los elementos de un documento HTML.

  • Se denota por un “*”

Selectores tipo Etiqueta

Selectores tipo etiqueta o de elementos, son los más simples y comunes

Selectores de Clase

Se utiliza para especificar un estilo a un grupo de elementos con la misma clase.

Su principal característica, es que se identifica por un punto “.”

Selectores de ID

Se utiliza para especificar un estilo a un elemento mediante su id.

Su principal característica es que se identifica por un sharp o gato “#”

Diferencia entre Clase e ID

Clase: se utiliza para un grupo de elementos, es decir como el apellido de nuestros elementos.

ID: se utiliza para un único elemento, no se repite ese id, es decir, como un numero de INE o CURP de nuestro elemento.

Selectores Descendientes

Afectan a elementos que están contenidos dentro de otro(s) elementos. Lo usamos mediante el uso de un espacio entre cada elemento.

Selectores de Hermanos

Los selectores de hermanos se emplean para seleccionar elementos adyacentes, es decir que su elementos padre es el mismo, y ambos elementos están al mismo nivel.

Hay dos tipos de selectores hermanos:

  • Selector Directo

    • Se utiliza cuando el hermano esta inmediatamente después de él.

    Lo usamos mediante el símbolo “+”

  • Selector General

    • Se utiliza cuando el hermano esta después de él. Lo usamos con el símbolo “~”

    Uso de CSS transforms:

    Al modificar las coordenadas del espacio, las transformaciones CSS permiten cambiar la posición del contenido afectado sin interrumpir el flujo normal. Se implementan haciendo uso de un conjunto de propiedades CSS que permiten aplicar transformaciones lineales a elementos HTML. Estas transformaciones incluyen rotar, torcer, escalar y desplazar en plano o en un espacio 3D.

    Propiedades de las transformaciones CSS

    Se usan dos propiedades principalmente para definir las transformaciones CSS:

    transform

    Especifica la transformación a realizar sobre el elemento. Se define como una lista separada por espacios de transformaciones que se aplican en orden una después de la otra, tal y como es requerido por la operación de composición.

    transform-origin

    Especifica la posición de origen que se usará para la transformación. Por defecto es el lado superior izquierdo del elemento. Esta posición es usada por varias transformaciones como rotar, escalar o torcer ya que necesitan de este punto especifico como parámetro.

    Propiedad @keyframes

    Propiedad @keyframes y animation

    Ambas propiedades trabajan de la mano, es decir una no puede trabajar si la otra no existe.

    Es importante saber que estas dos propiedades requieren del uso de prefijos.

    Propiedad @keyframes:

    • From

      • Estado inicial de la animación

    • to

      • Estado final de la animaciób

    • 0%

      • Estado inicial de la animación

    • 100%

      • Estado final de la animación

    • x%

      • Estado de la animación en un porcentaje x de tiempo

    Duplicar resolución

    Si existen varios conjuntos de keyframe para un nombre dado, el último encontrado se utiliza. Las reglas de @keyframes no estan en cascada, por lo que nunca las animaciones se conducen en keyframe desde un conjunto de reglas más de uno.

    Si una animación en tiempo compensado dado se duplica, el último keyframe en la regla de @keyframes para que ese porcentaje se utilice para ese frame. No hay una cascada dentro de una regla de @keyframes si varios kayframes especifican los valores de los mismos porcentajes.

    ¿Qué es el diseño responsivo?

    Diseño responsivo:

    El diseño web responsive o adaptativo es una técnica de diseño web que busca la correcta visualización de una misma página en distintos dispositivos. Desde ordenadores de escritorio a tablets y móviles.

    Hoy en día accedemos a sitios web desde todo tipo de dispositivos; ordenador, tablet, smartphone… por lo que, cada vez más, nos surge la necesidad de que nuestra web se adapte a los diferentes tamaños de los mismos.Pero, ¿qué es esto exactamente?

    Fluid Grid – Unidades relativas

    Imágenes flexibles – Unidades relativas

    Media Queries – Diferentes CSS

    Diseño responsivo – Ingredientes

    En el diseño responsivo haremos mucho uso de medidas de porcentaje o medidas relativas tales como “em” , “%”, etc… Aun así es válido poder trabajar con pixeles.

    ¿En qué consiste el diseño responsive?

    Se trata de redimensionar y colocar los elementos de la web de forma que se adapten al ancho de cada dispositivo permitiendo una correcta visualización y una mejor experiencia de usuario. Se caracteriza porque los layouts (contenidos) e imágenes son fluidos y se usa código media-queries de CSS3. El diseño responsive permite reducir el tiempo de desarrollo, evita los contenidos duplicados, y aumenta la viralidad de los contenidos ya que permite compartirlos de una forma mucho más rápida y natural.
